# Smart Canteen System

A simple Java console-based mini project developed to understand and practice Object-Oriented Programming (OOP) concepts using a real-world canteen ordering system.

## Features
- Interactive menu-driven program
- View food menu
- Add items to cart
- View order summary
- Bill calculation
- UPI and Cash payment methods

## OOP Concepts Used
- Encapsulation
- Inheritance
- Abstraction
- Polymorphism

## Technologies Used
- Java
- ArrayList
- Scanner Class

## Purpose
This project is created for learning Java OOP concepts in a simple and practical way.  
It is a beginner-friendly academic mini project and not a full-scale application.

## How to Run

### Compile
```bash
javac SmartCanteenSystem.java
```

### Run
```bash
java SmartCanteenSystem
```
